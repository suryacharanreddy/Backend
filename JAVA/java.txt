Identifiers:-
  1.Identifier should not start with numbers.
  2.Identifier should not contain any special symbols.
  3.Keyword/Reserve words should not used as Identifier.
  4.Identifier should only contain A-Z, a-z, 0-9, _, $.
  5.Case sensitive.
